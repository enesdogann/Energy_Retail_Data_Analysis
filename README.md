End-to-end electricity retail data analysis using **Python**, **Pandas** and **Matplotlib**. This project explores large-scale electricity billing and collection data to identify consumption patterns, customer behaviors and business insights.

---

# 📌 Project Overview

This case study was developed as part of the **Ahmet Çalık Foundation Advanced Data Analytics Training Program**.

The project focuses on analyzing electricity billing and collection datasets through exploratory data analysis, visualization and business-oriented reporting.

---

# 🎯 Objectives

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization
- Customer Segmentation
- District-based Consumption Analysis
- Collection Performance Analysis
- Business Insights & Recommendations
